Rick and Morty REST API

Initially, I wanted to make a page and have the user access it to learn more about the characters; however, it felt too complicated, so instead, I changed it to users filtering characters based on their gender, status, and name.

To ensure that I have gone above and beyond, I have dynamically created a button through JS. Although I had tried to ensure that the last page did not have a button, it was giving more issues, so the buttons still exist even if it is the last page. When a user types an invalid search, I ensure no buttons.

I had chosen this API as this is my favorite show, and I wanted to pay homage to it. Another thing I have added is a scrollable background image to make the website look more professional.

On top of that, I have ensured that users can type names in any way they want. However, the API only takes names starting with an uppercase letter; the rest are lowercase. To ensure that the results are still valid, I parsed usersâ€™ searches to match the way the API would like.
